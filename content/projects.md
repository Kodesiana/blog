## Kategori Fase

- **Production**, aplikasi dapat digunakan oleh pengguna serta data dan fitur aplikasi diperbarui secara berkala
- **Maintenance**, aplikasi dapat digunakan oleh pengguna, tetapi tidak ada fitur baru dan data tetap diperbarui secara berkala
- **End-of-Life**, akses ke aplikasi tidak dijamin, beberapa aplikasi mungkin masih bisa diakses tetapi admin tidak memberi jaminan bahwa aplikasi bisa digunakan semestinya, tidak ada fitur maupun pembaruan data lagi
